Пара слов по реализации чтобы просто понимать, что здесь происходит

Существует структура `Server` с внутренним хранилищем `Vault` и двумя каналами `receiver` и `exporter`.

У сервера есть два метода, вызываемых удаленно: Save и Export.

Метод Save запускает процесс вычитывания из канала и сохранения значений в хранилище.

Метод Export фактически делает обратное: достает данные из мапы и складывает их в канал.

Также существует `Clietn`, который умеет асинхронно рассылать данные по серверам. Он может использовать `SendTo` чтобы послать сообщение конкретному серверу, `BroadcastData`, чтобы отправить всем серверам, либо  `SplitSend` чтобы осуществить отправку по системе round robin

У него также есть метод `Receive`: он отправляет команду в специальный канал на сервере, вычитыв который, сервер начинает отправлять данные.
